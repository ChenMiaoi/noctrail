# Code of Conduct

Noctrail aims to be a direct, technical, and respectful project.

Expected behavior:

- Be precise and constructive in technical disagreement.
- Assume contributors are trying to improve the project.
- Keep discussion focused on code, design, safety, and user impact.
- Accept maintainer decisions after the tradeoffs have been discussed.

Unacceptable behavior:

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without consent.
- Repeatedly derailing technical discussions after a maintainer decision.

Maintainers may remove comments, close issues, or restrict participation when behavior harms the project.
