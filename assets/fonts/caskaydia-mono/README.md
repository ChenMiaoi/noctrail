Bundled `Caskaydia` mono font set for Noctrail.

Files:
- `CaskaydiaMonoNerdFontMono-Regular.ttf`
- `CaskaydiaMonoNerdFontMono-Bold.ttf`
- `CaskaydiaMonoNerdFontMono-Italic.ttf`
- `CaskaydiaMonoNerdFontMono-BoldItalic.ttf`

Runtime family name:
- `CaskaydiaMono NFM`

Purpose:
- These four static faces are the default primary font shipped by Noctrail.
- CJK and emoji still rely on system fallback fonts.

Provenance:
- The current copies were taken from the local Windows font installation under `C:\Windows\Fonts\`.
- Upstream design lineage is Microsoft's Cascadia font family, with Nerd Font symbol coverage in the `Caskaydia` mono naming used by the shipped files.
